**第2周学习路线（数组与对象）**

- 设备与时间：Node 18+、VS Code；建议本周安排 2–3 次、每次 60–90 分钟。
- 目标：掌握 map/filter/reduce/find，会遍历/解构对象，熟练用模板字符串输出结果。
- 产出：一份“学生成绩统计”小脚本，包含关键注释；执行后能打印统计信息。

**学习步骤（按顺序走）**

- 预热复盘（10 分钟）：回顾数组基本操作（push/pop/forEach）与 const 引用语义，准备一组学生对象样例。
- 进阶数组（30–40 分钟）：分别用 map（映射姓名/分数格式化）、filter（筛及格线，如 ≥60）、find（找到最高分/指定学号）、reduce（累计总分/平均分）。尝试链式组合。
- 对象操作（20–30 分钟）：用 for...in、Object.keys/values/entries 遍历；练习解构（const {name, score}）和深浅拷贝区别。
- 模板字符串（10 分钟）：用反引号和占位符 ${} 输出多行报告；对比字符串拼接的可读性。
- 综合练习（30–40 分钟）：完成“学生成绩统计”脚本：输入学生数组→输出总人数、平均分、及格率、最高/最低分名单；打印一张格式化表格。
- 自测与注释（15 分钟）：对每个核心步骤写一句注释；用 2–3 组测试数据（含边界：空数组、全满分、全不及格）验证输出。

**实践提示**
- 数据样例可放 Web/JavascriptDemo/week2/gradeStats.js；先写静态数组，后再考虑从 JSON 读取。
- reduce 初始值要写清楚，避免空数组报错；对象遍历时过滤掉原型属性（结合 hasOwnProperty 或 Object.keys）。
- 模板字符串里善用 \n 与对齐（如 padEnd/padStart）让表格易读。
- 记录收获与踩坑：本周至少写 5 点心得 + 1 个问题，存到同目录 notes.md。
