- **在VSCode中推荐的插件**
为提升 React 开发体验，推荐安装以下插件：
1. ES7+ React/Redux/React-Native snippets - React 代码片段
2. Prettier - Code formatter - 代码格式化
3. ESLint - 代码规范检查
4. Auto Rename Tag - 自动重命名 HTML 标签
5. Path Intellisense - 路径自动补全
6. Debugger for Chrome - Chrome 调试器

- **Vite React 项目结构**
```
my-react-app/
├── node_modules/        # 项目依赖
├── public/             # 静态资源
├── src/                # 源代码
│   ├── assets/         # 图片、字体等资源
│   ├── components/     # React组件
│   ├── App.jsx         # 根组件
│   └── main.jsx        # 入口文件
├── .gitignore          # Git忽略文件
├── index.html          # HTML模板
├── package.json        # 项目配置
└── vite.config.js      # Vite配置
```
